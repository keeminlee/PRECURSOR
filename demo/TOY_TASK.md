# Toy Demo Task

Use this task to see Precursor's Plan → Execute → Closeout workflow in action.

---

## The Task

**Write a plain-English Quick Start guide for PRECURSOR** that explains Plan, Execute, and Closeout with one worked example.

The guide should:
- Be understandable by someone who has never used Copilot agent workflows
- Explain what each phase (Plan, Execute, Closeout) does in 1–2 sentences
- Include one concrete example showing the full cycle
- Be a single markdown file, under 60 lines

---

## How to Run This Demo

1. Select **@PrecursorPlan** from the Copilot agent dropdown
2. Paste the task above (or just type: `Write a Quick Start guide for PRECURSOR that explains Plan, Execute, and Closeout with one worked example`)
3. Review the plan it creates
4. Select **@PrecursorExecute** and run step 1
5. Review the output

See [EXAMPLE_PLAN.md](EXAMPLE_PLAN.md) for what good Plan output looks like.
See [EXAMPLE_EXECUTE.md](EXAMPLE_EXECUTE.md) for what good Execute output looks like.
