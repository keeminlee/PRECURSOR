# RECURSOR — Quickstart

Extract. Open. Run `@Start`. You're operational.

---

## Setup (≤10 steps)

1. **Prerequisites:** VS Code with the GitHub Copilot extension (chat enabled). Sonnet or Opus model access required.
2. **Extract** the ZIP into an empty workspace folder — or into the root of an existing repo.
3. **Open** the folder in VS Code.
4. **Verify** that `RECURSOR/SKILLS/` and `.github/agents/` directories exist.
5. **Open Copilot chat** and type: `@Start`
6. `@Start` detects no `PROJECTS.md` and runs **CONFIG_PROJECTS** automatically — it scans your workspace and builds a project registry.
7. **Review** the generated `RECURSOR/PROJECTS.md`. Confirm the project list is correct.
8. **Done.** You're operational. `@Start` runs your morning coffee daily.
9. For a deeper walkthrough of agents and the daily loop: **`Run FIRST_BREW`**
10. For the full setup and verification flow: **`Run BOOTSTRAP`**
